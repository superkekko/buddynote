# buddynote
Buddynote is a simple personal markdown note manager

## Install and upgrade
### Requirements
- apache 2.4 (or Nginx)
- php 8.0

### Installation
Copy the files in the folder and give access to public. Default user is superadmin/superadmin

### Updates
Just copy fhe new version files
